- CONNECT 4 -

This is a 8-bit terminal-based game coded with python.
It has 4 diffrent levels that all have diffrent features.

- HOW TO RUN THE PROGRAM - 

It works using Python 3.8 or other versions and you have to run it in a terminal to play.
Typing the name of the folder and the type :   python connect4.py

1. Download the project folder

2. Open a terminal

3. GO to the folder containing the file 

4. Run the program

- HOW TO COMPLIE - 
This program does NOT require compilation because Python is an interpreted language.


- HOW TO PLAY - 

when the program is launched, select Start Game, Instructions or Quit.
Then choose a difficulty, read the instuctions again and play.
The goal is to connect pieces in a row (rules vary by level).
